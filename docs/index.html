Quickstart

From install to source results.

Ebisu turns a plain role and location into a focused Google search across supported ATS pages.

  1. 1Pin Ebisu in Chrome.After installing, open Chrome's Extensions menu and pin Ebisu so the popup is one click away.
  2. 2Search from the popup.Enter senior software engineer, choose Remote - United States, then click Go.
  3. 3Review source pages.Google opens ATS-hosted results. Ebisu adds Seen, Saved, and Applied controls where supported.
Pin in Chrome

Open Chrome's Extensions menu, then click the pin icon next to Ebisu so the popup stays in your toolbar.
